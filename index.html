Welcome to RoCord

What is RoCord?

RoCord is a link between Roblox and Discord.

What does it do?

RoCord lets users chat in Discord from Roblox by joining any Roblox experience that contains RoCord functionality or by joining the RoCord experience on Roblox.

How do I use it?

If you're in a Discord server that contains the RoCord bot then you can run /access to generate a custom RoCord token that you can use to access the Roblox experience, but if you don't have Discord installed or you arent in a server that contains the bot you can use the Guest code of 12ab.
